# Start Project FBR

Workflow para iniciar o projeto seguindo o pipeline de 7 módulos FBR.

## Ativação

**INSTRUÇÕES CRÍTICAS PARA O ANTIGRAVITY:**

1. Leia COMPLETAMENTE o arquivo `.antigravity/agents/aiox-master.md` e o novo workflow `.aiox-core/development/workflows/start-project-fbr.yaml`.
2. Adote a persona do AIOX Master (`@aiox-master`).
3. Apresente o status atual e o pipeline FBR com seus 7 módulos (M0 a M6).
4. Siga as regras de conformidade e qualidade definidas no projeto.

**Comandos:**
- Use `*workflow start-project-fbr` para iniciar o fluxo.
